# Мини-Игра "Угадай число" на основе TCP сервера ZeroMQ

## Требования:
1. NodeJS поддерживаемой версии
2. ZeroMQ (устаналивается полу-автоматически, см. пункт 2)

## Установка:
1. Клонируйте (установите) этот репозиторий
2. Установите зависимости - npm install

## Использование:
1. Запустите сервер (game-server.js).
Примечание: В случае успешного запуска в консоли отобразится сообщение "Готов к игре..."
2. Запустите клиент (game-client.js), указав при запуске диапазон чисел.
Примечание: В случае успешного запуска в консоли отобразятся сообщения с указанием диапазона чисел и загаданным клиентом числом.

### Пример использования:
1. node game-server.js - запуск сервера
2. node game-client.js 1 100 - запуск клиента, где 1 и 100 это диапазон чисел.

## Дополнительная информация:
1. Диапазон загадываемых чисел должен состоять из целых, положительных чисел.
2. В случае бездействия клиента после его запуска - убедитесь, что сервер запущен.
3. Все действия как клиента, так и сервера доступны в консоли в формате логов.
